🌤 Weather Bot

Телеграм-бот прогноза погоды: текущая погода, почасовой прогноз и прогноз на 5 дней для любого города мира.

Живой бот: [@weatherbergov_bot](https://t.me/weatherbergov_bot)

 Возможности

| | |
|---|---|
| 🔍 | Погода по названию города — кириллица и латиница |
| 📍 | Определение города по геолокации |
| 📊 | Почасовой прогноз на сутки + график температуры |
| 📅 | Прогноз на 5 дней с мин/макс температурой |
| ⭐️ | Избранные города — у каждого пользователя свои |
| 🔎 | Inline-режим: работает в любом чате — `@weatherbergov_bot Москва` |

---

Стек

- **Python 3.11**
- **aiogram 3.x** — асинхронный фреймворк для Telegram Bot API
- **aiohttp** — HTTP-клиент к OpenWeatherMap API
- **aiosqlite** — асинхронная работа с SQLite

---

 Технические решения

**Изоляция слоёв.** Класс `WeatherService` инкапсулирует всю работу с внешним API — хендлеры Telegram ничего не знают об HTTP. Сервис прокидывается в хендлеры через DI (`Dispatcher(weather_service=...)`), без глобальных переменных.

**Кэширование.** Ответы API кэшируются на 10 минут (TTL). Повторные запросы того же города не тратят лимит бесплатного тарифа OpenWeatherMap.

**Агрегация данных.** OpenWeatherMap отдаёт прогноз с шагом 3 часа. Дневной прогноз собирается на стороне бота: интервалы группируются по датам, вычисляются мин/макс температуры, доминирующее состояние погоды определяется по частоте.

**Обработка ошибок.** Явно обрабатываются: город не найден (404), невалидный ключ (401), превышение лимита (429), сетевые сбои. Пользователь получает понятное сообщение, бот не падает.

**Визуализация без библиотек.** График температуры рисуется юникод-символами (`▁▂▃▄▅▆▇█`), полоса светового дня — позицией солнца между рассветом и закатом.

## Запуск

```bash
git clone https://github.com/USERNAME/telegram-weather-bot.git
cd telegram-weather-bot

pip install -r requirements.txt

cp .env.example .env
# заполнить .env своими ключами

python main.py
```

### Переменные окружения

| Переменная | Описание | Где взять |
|---|---|---|
| `BOT_TOKEN` | Токен Telegram-бота | [@BotFather](https://t.me/BotFather) → `/newbot` |
| `WEATHER_API_KEY` | Ключ OpenWeatherMap | [openweathermap.org](https://openweathermap.org/api) — бесплатный тариф |

> Новый ключ OpenWeatherMap активируется до 2 часов. До этого API возвращает 401.

### Настройка бота

Inline-режим включается у [@BotFather](https://t.me/BotFather): `/setinline` → выбрать бота → задать текст-подсказку.
